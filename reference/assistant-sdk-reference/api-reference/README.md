# API Reference

The Assistant SDK provides two primary APIs:&#x20;

* [Assistant API](assistant.md) - manage active assistant session, access combined audio and other in room actions
* [Trigger API](trigger.md) - programmatically control assistants using [webhooks](../../../whereby-product-features/insights-suite-and-api/webhooks.md)

There can be used in tandem to create a rich, automated Assistant experience. The details of each API are outlined in the sections below.
