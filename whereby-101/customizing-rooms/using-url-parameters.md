# Using Attributes/URL Params

Attributes can be used on Whereby's pre-built Web Component by adding them to the \<whereby-embed> element.&#x20;

{% hint style="info" %}
Note that Attributes and URL Parameters are _only_ compatible with Whereby's pre-built UI and will not work with the React SDK.
{% endhint %}

For example, the following component would open the room with the screenshare and people buttons hidden:

<pre class="language-html" data-overflow="wrap" data-line-numbers><code class="lang-html">&#x3C;whereby-embed 
<strong>    room="https://subdomain.whereby.com/your_room?roomKey=3fe345a"
</strong>    screenshare=off
    people=off
/>
</code></pre>

URL Parameters are used with an iframe implementation and appended directly to the roomUrl. Several parameters can be combined by using the ampersand symbol (&).&#x20;

For instance, the following iframe implementation uses URL parameters to disable the screenshare and people buttons:

{% code overflow="wrap" lineNumbers="true" %}
```html
<iframe 
    src="https://subdomain.whereby.com/room?screenshare=off&people=off"
    allow="camera; microphone; fullscreen; display-capture; autoplay; compute-pressure"
/>
```
{% endcode %}
